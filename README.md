# Vue 3 + TypeScript + Vite + Vuetify3 Template

## Description

This is a template project for Vue 3 + TypeScript + Vite + Vuetify3 + Axios + Vue Router + Pinia + ESLint + Prettier.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev
```

```bash
# production mode
$ yarn build
```

## Linting and formatting

```bash
# lint
$ yarn lint
```

```bash
# format
$ yarn format
```
